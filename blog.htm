Minha Jornada da Infraestrutura para Programação

15 de Março, 2024 5 min de leitura 127 visualizações 23 curtidas
carreira transição HTML CSS JavaScript

Há 6 meses, eu estava trabalhando com infraestrutura e monitoramento de sistemas na Brisanet, mas sentia que precisava de algo mais. A programação sempre me intrigou, e decidi que era hora de dar o próximo passo.

O Início da Transição

Meu primeiro contato com código foi através de HTML e CSS. Inicialmente, parecia simples demais, mas rapidamente percebi o poder que essas tecnologias têm. Comecei construindo páginas simples, depois fui adicionando JavaScript para torná-las interativas.

Aprendizados Fundamentais

1. HTML e CSS não são "simples"

Muitos subestimam o poder de HTML e CSS bem escritos. Aprendi que:

  • Semântica é crucial para acessibilidade
  • CSS Grid e Flexbox são ferramentas poderosas
  • Performance é tão importante quanto estética

2. JavaScript é o que dá vida

Quando comecei com JavaScript, percebi que podia criar experiências verdadeiramente interativas. Meus primeiros projetos incluíram:

// Exemplo de validação de formulário
function validarEmail(email) {
    const regex = /^[^\s@]+@[^\s@]+\.[^\s@]+$/;
    return regex.test(email);
}

// Uso
const email = "usuario@exemplo.com";
console.log(validarEmail(email)); // true

3. A lógica vem com a prática

Vim da infraestrutura, então já tinha uma base de pensamento lógico. Isso me ajudou muito na transição, especialmente quando comecei com:

  • Loops e condicionais
  • Funções e escopo
  • Manipulação de DOM

O Próximo Passo

Agora estou focado em:

  • React - Para construir aplicações mais complexas
  • Node.js - Para desenvolvimento backend
  • Python - Para automação e análise de dados

Conclusão

A transição de carreira não é fácil, mas é extremamente recompensadora. Cada linha de código que escrevo me aproxima mais do meu objetivo de me tornar um desenvolvedor full-stack.

Dica para quem está começando: Não tente aprender tudo de uma vez. Foque em construir projetos reais, mesmo que simples. A prática é o melhor professor.
