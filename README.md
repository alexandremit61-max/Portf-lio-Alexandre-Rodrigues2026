# 🚀 Portfolio Profissional - Alexandre Rodrigues.

## 📋 Sobre o Projeto

Portfolio profissional completo desenvolvido para destacar a transição de carreira de Alexandre da Silva do mundo de infraestrutura de TI para desenvolvimento web. O projeto demonstra habilidades técnicas, projetos realizados e pensamentos sobre tecnologia através de um blog integrado.

## ✨ Funcionalidades Principais

### 🎨 Interface Avançada
- **Animações de Partículas**: Sistema de partículas interativas no fundo
- **Animação de Digitação**: Texto dinâmico com efeito de máquina de escrever
- **Modo Claro/Escuro**: Alternância suave entre temas
- **Animações de Scroll**: Efeitos visuais ao rolar a página
- **Design Responsivo**: Adaptação perfeita para dispositivos móveis

### 📝 Sistema de Blog Completo
- **CRUD Dinâmico**: Criar, ler, atualizar e deletar posts
- **Busca e Filtros**: Sistema de busca com filtros por categoria
- **Visualização em Modal**: Leitura de posts em modal expansivo
- **Curtidas e Compartilhamento**: Interações sociais com os posts
- **Editor com Preview**: Edição de posts com preview em tempo real

### 📊 Dashboard Interativo
- **Gráficos em Tempo Real**: Chart.js e ECharts com atualizações automáticas
- **KPIs Animados**: Indicadores com animações de contador
- **Múltiplos Gráficos**: Tráfego, performance, skills, atividades
- **Filtros por Período**: Visualização de dados por diferentes períodos

### 🏆 Sistema de Certificados
- **Catálogo de Certificados**: Visualização organizada por categorias
- **Modal de Detalhes**: Visualização detalhada de cada certificado
- **Filtros e Busca**: Sistema de filtragem por categoria e busca textual
- **Verificação Online**: Links para verificação dos certificados

### 🎯 Sistema de Conquistas
- **Gamificação**: Sistema de conquistas com níveis de raridade
- **Animações**: Efeitos visuais ao conquistar novos itens
- **Progresso Visual**: Representação visual do progresso pessoal

## 🛠️ Tecnologias Utilizadas

### Frontend
- **HTML5**: Estrutura semântica e acessível
- **CSS3**: Animações avançadas, Grid, Flexbox, Custom Properties
- **JavaScript ES6+**: Classes, async/await, fetch API
- **Chart.js**: Gráficos interativos e animados
- **ECharts**: Gráficos avançados e personalizados

### Bibliotecas e Frameworks
- **Font Awesome**: Ícones profissionais
- **Google Fonts**: Tipografia Inter
- **Particles.js**: Sistema de partículas
- **Typed.js**: Animação de digitação
- **Animate.css**: Animações pré-definidas

### Backend (Simulado)
- **RESTful API**: Sistema de tabelas para persistência de dados
- **LocalStorage**: Armazenamento de preferências do usuário

## 📊 Dashboard Analytics

O dashboard inclui:
- **Gráficos em Tempo Real**: Atualizações automáticas a cada 5 segundos
- **KPIs Animados**: Contadores com animações suaves
- **Múltiplos Gráficos**: Tráfego, performance, skills, atividades
- **Filtros por Período**: 24h, 7d, 30d
- **Tema Adaptativo**: Cores que mudam com o tema

## 📝 Blog System

O sistema de blog possui:
- **CRUD Completo**: Criar, editar, deletar posts
- **Busca e Filtros**: Por categoria e texto
- **Visualização Modal**: Leitura em modal expansivo
- **Curtidas**: Sistema de curtidas por post
- **Editor com Preview**: Preview em tempo real

## 🏆 Certificates System

O sistema de certificados inclui:
- **Catálogo Organizado**: Por categorias (Frontend, Backend, etc)
- **Modal de Detalhes**: Visualização completa do certificado
- **Filtros e Busca**: Por categoria e texto
- **Verificação Online**: Links para verificar autenticidade
